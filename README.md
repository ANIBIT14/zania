## Getting Started

First, install the dependencies using but ensure that you are using node version over 181.18.0:

```bash 
npm install --legacy-peer-deps


Then, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
